## Features

## Installation

1. Clone this repository

```bash
git clone https://github.com/aditya8Raj/react-todo.git
```

2. Install all dependencies

```bash
npm install
```

3. Run the server

```bash
npm run dev
```

## Usage

1. Open your browser and go to `http://localhost:5173/`
